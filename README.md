
## About
* Built on:
  * php 7.4+
  * Laravel 8
* Uses [Laravel/jetstream](https://jetstream.laravel.com/2.x/introduction.html)
  * with [Livewire](https://laravel-livewire.com/)
* Comes pre-scaffolded with user roles, and permissions
* `ddf()` and `ddq` for easy debugging
* Enabled [Laravel IDE helper](https://github.com/barryvdh/laravel-ide-helper)
* Enabled [Laravel Telescope](https://laravel.com/docs/8.x/telescope)
* Added [Alpine.js](https://github.com/alpinejs/alpine/)
